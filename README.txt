testing use of C++ FLTK GUI lib from D code.
fltkLib is the C++ GUI using FLTK (fluid)

The GUI is defined in C++ using FLTK Fluid, in a class that implements an Interface paralled in C++/D,
the C++ GUI is accessed through this interface and vice-versa for the C++ code calling D functions.
